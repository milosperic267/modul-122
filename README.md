## Beschreibung
Mein Skript solle folgendes Problem lösen. 

Wir laden täglich Dateien und Programme aus dem Internet hinunter. Diese Dateien werden im Downloads-Folder gespeichert. Wenn man diesen Ordner nicht regelmässig löscht, sammeln sich schnell mal sehr viele Dateien an.

Die Lösung zu diesem Problem ist mein Bash-Abschlussprojekt. Dieses Bash-Programm fragt den User, ob er den Downloads-Ordner löschen will. Der User gibt "j"/"J" oder "n"/"N" ein. Wenn die Eingabe des Users "j"/"J" ist, wird der gesamte Downloads-Ordner unverzüglich gelöscht. Falls die Eingabe "n"/"N" ist wird das Programm beendet und der Ordner wird nicht gelöscht. Wenn die Eingabe ungültig ist, wird der User dazu aufgefordert eine gültige Eingabe zu geben.

## Testing


| Eingabe  | Resultat |
| ------------- | ------------- |
| "j"/"J"  | Programm navigiert in den Download-Ordner und löscht unverzüglich den ganzen Inhalt des Ordners  |
| "n"/"N"  | Programm wird abgebrochen und der Download-Ordner wird nicht gelöscht.  |
| ungültige Eingabe  | Programm fordert User auf eine gültige Eingabe einzugeben.  |
